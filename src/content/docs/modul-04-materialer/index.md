Gweike LC1390N er designet for kutting og gravering av **ikke-metalliske materialer**. Det er avgjørende å bruke riktige materialer — feil valg kan forårsake giftige gasser, brann eller skade på maskinen.

![Materialer som ikke må kuttes](/img/materialer-forbudt.png)

<Aside type="danger">
Ved **usikkerhet**: Spør en ansvarlig før du setter i gang. Det er alltid bedre å spørre enn å risikere skade på helse eller utstyr.
</Aside>

## Tillatte materialer

| Materiale | Egnet for | Innstillinger | Merknad |
|---|---|---|---|
| **Akryl (Plexiglass)** | Kutt, gravering | Høy effekt, middels hastighet | Ren kant, god finish |
| **MDF** | Kutt, gravering | Middels effekt | Gir mye røyk — krever godt avtrekk |
| **Kryssfinér** | Kutt, gravering | Middels effekt | Sjekk limtype — noen lim avgir gasser |
| **Lær (ekte/kunstig)** | Kutt, gravering | Lav-middels effekt | Ekte lær lukter sterkt — ekstra ventilasjon |
| **Papir og papp** | Kutt, gravering | Lav effekt | Høy brannfare — vær ekstra oppmerksom |
| **Tekstiler (bomull, polyester)** | Kutt, gravering | Lav-middels effekt | Kan ta fyr lett — aldri ubemannet |
| **Papir** | Kutt | Lav effekt | Tynt papir krever lav effekt |

## Forbudte materialer — og hvorfor

| Materiale | Årsak til forbud |
|---|---|
| **PVC (polyvinylklorid)** | Avgir **klorholdig gass** som er dødelig farlig og korroderer maskinen |
| **Vinyl** | Samme problem som PVC — kloravgass |
| **ABS (plast)** | Avgir **cyanidholdig gass** (giftig) og svært ujevn flamme |
| **Polykarbonat (Lexan)** | Gir dårlig kutt, avgir skadelige gasser, kan smelte og antenne |
| **Isopor (EPS)** | Smelter umiddelbart, høy brannfare, giftige avgasser |
| **Glass** | CO2-lasere kan ikke kutte glass effektivt — risikerer sprengning |
| **Metall (tykt)** | LC1390N er ikke designet for metallkutting |

<Aside type="danger">
**Unntak for metall:** Maskinen kan kutte/gravere **tynn rustfritt stål og karbonstål** (ca. 0,5 mm) ved gravering med riktige innstillinger. Men den kan **IKKE** kutte tykkere metaller. Overskridelse kan skade laserrøret og forårsake brann.
</Aside>

## Når du er usikker

1. **Sjekk materialets datablad** — Se etter klorinnhold, giftige tilsetninger
2. **Spør en ansvarlig** — Det finnes ingen dumme spørsmål når det gjelder sikkerhet
3. **Test på liten prøve først** — Aldri start med full størrelse på ukjent materiale
4. **Lukten er en indikator** — Sterk, unaturlig lukt betyr at du bør stoppe umiddelbart
